## Selected Projects

### VerdictCore — Deterministic AI Decision Engine

A Python-based decision framework for building structured, explainable, audit-ready decision engines where deterministic rules select outcomes and LLMs assist with narrative, evidence extraction, and explanation.

**Highlights**

- Formal decision objects instead of ad-hoc prompts
- Deterministic scoring, constraints, ranking, and canonical outputs
- Explainability: why selected, why not, top drivers, sensitivity
- Audit hashing, reproducibility, governance, policy drift detection
- Monte Carlo simulation, stress testing, Pareto trade-offs
- Human review, override, decision registry, and adaptive memory

**Tech:** Python, Pydantic, Typer CLI, SQLite, YAML, JSON, simulation, optimization, governance logic  
**Repository:** [VerdictCore](https://github.com/aida-solat/verdictCore)

---

### Smart Logistics — Causal Decision Copilot

A causal, risk-aware logistics decision system that moves beyond prediction into prescriptive decision support.

**Highlights**

- Causal effect estimation using DoWhy / EconML
- CVaR optimization for inventory and routing decisions
- OR-Tools routing with Monte Carlo risk scoring
- SimPy digital twin for offline A/B validation
- SHAP explainability and causal path tracing
- LLM decision narration and operator Q&A
- Feedback loop for override logging and adaptive calibration

**Tech:** FastAPI, PostgreSQL, Redis, DoWhy, EconML, OR-Tools, CVXPY, SimPy, SHAP, MLflow, Prometheus, Grafana, Next.js  
**Repository:** [Smart Logistics](https://github.com/aida-solat/smart-logistic-api)

### Aegis — Rules-First AI Platform for Insurance Decisions

An auditable AI platform for insurance underwriting and claims where deterministic rule engines compute risk and fraud scores, while LLMs generate grounded explanations.

**Highlights**

- Rules-first decision architecture for regulated insurance workflows
- Deterministic risk and fraud scoring
- Swappable LLM layer: OpenAI, Anthropic, Ollama, heuristic fallback
- Append-only decision audit log
- Underwriting and claims decision endpoints
- Full-stack product surface with backend, frontend, CI, Docker, and deployment setup

**Tech:** FastAPI, SQLModel, Python, OpenAI, Anthropic, Ollama, Next.js, React, Tailwind, Docker, GitHub Actions  
**Repository:** [Aegis](https://github.com/aida-solat/Insurance-API-Marketplace)

### DecisionIQ — Multi-Model Decision Intelligence Platform

A full-stack decision intelligence platform for supply chain and real estate analytics.

**Highlights**

- Forecasting with XGBoost, LightGBM, LSTM, and Optuna tuning
- Anomaly detection using rolling z-score, Isolation Forest, and DBSCAN
- Monte Carlo risk scoring with VaR, percentile statistics, and sensitivity analysis
- ChromaDB-based RAG advisor with GPT-4o-mini fallback
- REST + SSE backend, async jobs, MLflow tracking, and Prometheus metrics

**Tech:** FastAPI, Next.js, PostgreSQL, Redis, Celery, MLflow, Prometheus, ChromaDB, XGBoost, LightGBM, LSTM  
**Repository:** [DecisionIQ](https://github.com/aida-solat/decision-iq)
